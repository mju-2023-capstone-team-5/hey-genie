## 🚙 사유 주차장 공유 어플리케이션, SOPAR
| Capstone Design Project
- <b>주차난 해결</b>을 위한 사유 주차장 공유 어플리케이션
- Capstone Design 금상 수상

## 🔗 주요 제공 기능
1. <b>위치 검색</b>으로 원하는 곳에 사유 주차장 공유 가능!
2. 예약 시간 10분 전 <b>푸쉬 알람</b>!
3. 간단하게 리뷰를 등록하고!
4. 주차장에 대한 전체 리뷰를 <b>ChatGPT를 통해 요약된 한문장</b>으로!
